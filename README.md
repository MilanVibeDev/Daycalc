# DayCalc

DayCalc is a simple web tool that answers one question: *"If today is X, what day will it be in N days?"* You pick the current day of the week, type in a number, hit Submit — and it tells you the answer. No fluff, no frills.

**Tech used:** HTML, CSS, JavaScript

**Features:**
- Day selector (Monday – Sunday)
- Number input for days ahead
- Instant result on submit

---

## 🛠 Process

DayCalc was my first project when I started learning web development back in 2023. I built it to prove something to myself — and to have a reference point for how far I'd come.

The idea came from a simple problem I kept running into mentally: trying to count days in my head. I started with the logic first — figuring out the modular arithmetic to wrap days of the week correctly — and then built a minimal UI around it. No design system, no framework, just raw HTML, CSS, and JS. It was rough, but it worked, and finishing it felt like a big deal at the time.

Looking back now, three years later, the gap between what I built then and what I'd build today is exactly why I've kept this up on GitHub.

---

## 🚀 How It Could Be Improved

- **Responsive design** — the layout isn't built for mobile at all; a proper responsive CSS pass would make it accessible on any screen
- **Better UX** — clearer input validation, error messages, and maybe a more intuitive UI overall
- **More features** — support for entering a specific date instead of just a day name, a reverse mode (how many days between two days?), or a countdown to a specific day of the week
